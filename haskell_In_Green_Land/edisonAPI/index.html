## Analyzing the Energy Behavior of a Purely Functional Language ### Edison API Below is a summary of the Edison API, for its various Abstractions. If you are interested in the full details head over to the hackage page, [here](http://hackage.haskell.org/package/EdisonAPI "Edison API."). Edison functions which are specific to Sets: * Constructors: insertWith, insertSeqWith, unionl, unionr, unionWith, unionSeqWith, fromSeqWith * Set operations: intersection, difference, symmetricDifference, properSubset, subset, intersectionWith -
